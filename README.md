1.Starting zapret

- Open the Zapret folder.

- Run service.bat.

- In the menu that appears:
- Press 4 to enable the game filter.

- Press 5 to select "any" for the IPSet Filter; if "any" does not appear, press 5 again.

- Press 1 and choose the working .bat file (preferably ALT11).

- Press 3 to check if everything works.

2. After setup

- You can close service.bat and then watch YouTube or play games with friends in discord.
  
- At every startup, service.bat will run in the background automatically.

This setup ensures proper filtering and VPN usage to allow Roblox, Discord, and YouTube to work correctly.

2(1). To start Discord and YouTube

- You only need to start Zapret.
