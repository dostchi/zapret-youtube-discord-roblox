If you need to start roblox you need to download CloudFlare WARP and start it with the bat, for discord and youtube you need just start zapret.
For start zapret you need to open zapret folder and start service.bat. Then press 4 for enable game filter, then press 5 to choose "any" on IPset Filter, if you don't see "any" next to IPSet Filter press 5 again. Then press 1 and choose working bat(ALT11), then press 3 to check if it works or not.
then you can close service.bat and watch youtube or play with friends))
at any start, service.bat will be launched in the background.
